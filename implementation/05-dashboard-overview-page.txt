2. DATA SOURCES
----------------

All data comes from Supabase via custom hooks:

  const { user } = useAuth();
  const { profile } = useProfile();
  const { campaigns, loading: campaignsLoading } = useCampaigns();
  const { aggregateMetrics, dailyMetrics } = useMetrics();

Queries:

  KPI Cards:
    - Active campaigns: SELECT count(*) FROM campaigns WHERE user_id = ? AND status = 'active'
    - Total spend: SELECT SUM(budget_spent) FROM campaigns WHERE user_id = ?
    - Total conversions: SELECT SUM(conversions) FROM campaign_metrics WHERE campaign_id IN (user's campaigns) AND date >= NOW() - '30 days'
    - Avg ROAS: SELECT AVG(roas) FROM campaign_metrics WHERE ... (weighted by spend)

  Performance Chart:
    SELECT date, SUM(spend), SUM(conversions), AVG(ctr), AVG(cpa)
    FROM campaign_metrics
    WHERE campaign_id IN (user's campaigns)
    AND date >= NOW() - interval '{period} days'
    GROUP BY date
    ORDER BY date ASC

  Recent Campaigns:
    SELECT * FROM campaigns WHERE user_id = ? ORDER BY updated_at DESC LIMIT 5

  AI Insights:
    Generated via Gemini using aggregated campaign data (see 03-gemini-api-integration.txt)


3. STAT CARD COMPONENT
------------------------

Reusable component: StatCard.tsx

Props:
  label: string        — "Active Campaigns"
  value: string        — "4" or "MYR 3,428"
  trend: number        — percentage change (+14 or -2)
  icon: LucideIcon     — Activity, DollarSign, Target, TrendingUp
  sparklineData?: number[] — 7 data points for mini chart

Features:
  - Large value in font-heading italic
  - Green trend arrow for positive, red for negative
  - Tiny SVG sparkline at bottom (7-day trend)
  - Hover: subtle lift + shadow increase
  - Loading state: skeleton pulse animation


4. PERFORMANCE CHART COMPONENT
--------------------------------

Component: PerformanceChart.tsx

Props:
  data: Array<{ date: string, spend: number, conversions: number, ctr: number, cpa: number }>
  activePeriod: "7d" | "14d" | "30d" | "90d"
  activeMetric: "spend" | "conversions" | "ctr" | "cpa"

Implementation options (in order of preference):
  
  Option A — Recharts (recommended):
    npm install recharts
    Use <AreaChart> with gradient fill
    Clean, minimal, matches our design system
    Responsive container

  Option B — Custom SVG:
    Calculate path from data points
    SVG <path> with gradient <linearGradient> fill
    Full design control, zero dependencies
    More work to implement tooltips and axis labels

  Option C — Chart.js via react-chartjs-2:
    Heavier bundle, canvas-based
    Less consistent with our SVG-first aesthetic
    Not recommended

Chart styling:
  - Gradient fill: slate-900 at 0.2 opacity → transparent
  - Line: 2px, slate-900
  - Grid: dotted, slate-100
  - Axis labels: font-body, text-xs, text-slate-400
  - Tooltip: bg-slate-900 text-white rounded-xl shadow-lg
  - Period tabs: pill-style toggle buttons

7. AI INSIGHTS PANEL
---------------------

Component: AIInsightPanel.tsx

Uses the Gemini generateInsights() function from lib/gemini.ts.

On mount (or when user clicks "Refresh"):
  1. Gather aggregated campaign data from useCampaigns() and useMetrics()
  2. Send to generateInsights()
  3. Display results as insight cards

Each insight card:
  - Left border color:
    - success (green): good performance
    - warning (amber): declining metric
    - opportunity (blue): actionable improvement
  - Icon matching the type
  - Title in font-body font-semibold
  - Description in font-body text-sm text-slate-500
  - Action text in font-body font-medium text-slate-800

Fallback if Gemini is unavailable:
  Show static placeholder insights based on hardcoded rules:
  - If ROAS > 3: "Strong ROAS performance across campaigns"
  - If any campaign paused: "Review paused campaigns for reactivation"
  - If email CTR < 2%: "Email open rates need attention"


8. LOADING STATES
------------------

  - Table: 5 skeleton rows with pulse
  - AI Insights: "Generating insights..." with spinner

Skeleton component: LoadingSkeleton.tsx
  Props: width, height, className
  Renders a div with bg-slate-100 rounded-lg animate-pulse
