## 🌟 About Veltrix

Veltrix is a next-generation marketing automation platform. Instead of manually writing copy, tweaking bids, and running A/B tests across different platforms, Veltrix uses advanced AI models to completely autonomous your campaign management across **Meta Ads, Google Search, X (Twitter), and Email**.

It learns from your audience, tests continuously, and scales the winning variations without you ever having to lift a finger.

## 🚀 Key Features

- **Multi-Channel Orchestration**: Launch unified campaigns across all major platforms in one click.
- **Autonomous A/B Testing**: The AI constantly generates and tests new copy and creative variations.
- **Continuous Bid Optimization**: Smart budget allocation ensures your money is spent where it yields the highest ROI.
- **Custom Brand Voice Models**: Fine-tune the AI on your brand's existing content so it always sounds like you.

## 💻 Tech Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS (Custom Glassmorphism Design System)
- **Deployment**: Vercel / Netlify (Ready)

## 🛠️ Quick Start

Follow these steps to run Veltrix locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nimalanrao/veltrix.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd veltrix
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
